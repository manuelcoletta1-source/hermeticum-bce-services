Percorso rapido (4 step)

Completa questi quattro passaggi per ottenere un set coerente e verificabile. Tutto client-side. Nessun upload.

  1. Create — genera ipr.json + receipt.json (firma ED25519).
  2. Verify — verifica locale (hash + firma). Fail-closed.
  3. Anchor — costruisci anchor-pack con riferimenti BTC/ETH/IPFS (manuale).
  4. Evidence Pack — crea PACK_MANIFEST firmato: IPR + receipt + anchors + proof.
