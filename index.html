Percorso rapido (4 step) — “fai solo questo”

Se completi questi quattro passaggi, ottieni un set coerente e verificabile. Ogni step è client-side. Nessun upload.

  1. Create — genera ipr.json + receipt.json (firma ED25519). Opzionale: SIGNING_KEYPACK.json.
  2. Verify — verifica locale (hash + firma ED25519). Controllo anchors se presenti. Fail-closed.
  3. Anchor — costruisci anchor-pack.json con riferimenti BTC/ETH/IPFS (manuale, nessuna promessa).
  4. Evidence Pack — crea PACK_MANIFEST.json firmato: IPR + receipt + anchor-pack + README proof.
