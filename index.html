EU Gateway

Piattaforma neutra per Create / Verify / Evidence Pack. Produce output verificabili offline (hash/receipt/manifest) in regime controllato.

UE-first GDPR-min Hash-only Audit-first Fail-closed

Crea un IPR (Create) Verifica (Verify) Evidence Pack

Create

Genera output minimo e verificabile:
• `ipr.canon.json`
• SHA-512
• `receipt.json` (ED25519, se previsto)
• `PACK_MANIFEST.json`

Apri Create

Verify

Verifica offline indipendente:
• hash → firma → manifest
Se una verifica fallisce o è indeterminata ⇒ invalid.

Regole globali

FAIL-CLOSED: mismatch o indeterminatezza ⇒ output invalidato e azione bloccata.

HASH-ONLY: nel gateway non transitano documenti, solo impronte verificabili.

AUDIT-FIRST: ogni output deve poter essere verificato offline da terzi.

Note (perimetro)
