Policy posture

EU-first — EU alignment is the default reference posture.
Audit-first — outputs are designed to be inspected and compared.
Hash-only — public verification uses cryptographic references, not raw identity data.
No data custody — no centralized custody of personal identity data.
Fail-closed — if verification is not possible, execution is treated as invalid.
