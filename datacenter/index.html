IPR DataCenter = esecuzione metrologica

L’IPR DataCenter è l’ambiente di esecuzione dell’infrastruttura HERMETICUM. Esegue operazioni solo se precedute da verifica ex-ante valida.

Non custodisce identità. Non introduce discrezionalità. Non esiste esecuzione senza PASS.

Comportamento

  • PASS → esecuzione consentita
  • FAIL → nessuna esecuzione
  • Nessun fallback permissivo
  • Nessuna custodia dati personali

Funzione nell’infrastruttura

Il DataCenter non decide. Applica esiti metrologici deterministici provenienti dal Gate.
