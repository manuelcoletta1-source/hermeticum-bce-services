Create

Genera un pacchetto IPR in modalità hash-only. Nessun documento richiesto. Output verificabile offline.

UE-first GDPR-min Hash-only Fail-closed Audit-first

Output (standard)

• ipr.canon.json (record canonicalizzato)
• SHA-512 del canonico
• receipt.json (firma ED25519 sul digest, se previsto)
• PACK_MANIFEST.json (indice dell’Evidence Pack)

Regola fail-closed

Se un campo minimo è mancante o non coerente, l’output va considerato non valido. La verifica deve poter fallire in modo deterministico.

Vai a Verify

Checklist rapida

1) Genera `ipr.canon.json` (minimo, deterministico)
2) Calcola SHA-512
3) Firma il digest (ED25519) → `receipt.json`
4) Crea `PACK_MANIFEST.json` (hash-only, append-only)

Note

Questo processo produce evidenza tecnica verificabile. Non sostituisce eIDAS/EUDI, non gestisce identità civile, non attribuisce effetti giuridici automatici.
