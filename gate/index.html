JOKER = interfaccia metrologica ex-ante

JOKER è l’interfaccia metrologica dell’infrastruttura HERMETICUM. Misura ex-ante le condizioni minime di ammissibilità di un’azione e restituisce un esito deterministico: PASS oppure FAIL.

Non è un’autorità. Non decide nel merito. Non custodisce identità. Non introduce sorveglianza. Opera come misura tecnica prima dell’esecuzione.

Verifica OpenAPI Esecuzione

Esito

  • PASS → esecuzione ammessa
  • FAIL → esecuzione bloccata (fail-closed)

Regola unica

In assenza di prova ex-ante → FAIL → nessuna esecuzione.
