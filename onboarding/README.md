# Onboarding — Livello 1 (BASE)

Questa cartella contiene gli artefatti standard per l’attivazione di una
BioCyber Identity Card — Livello 1 (BASE).

## Contenuto
- `manifest-level1.json` — manifest ufficiale IPR (machine-readable)

## Regole
- nessun dato personale nel manifest
- disclosure pubblica limitata agli hash
- timestamp certificato obbligatorio
- ancoraggi pubblici disattivati di default

## Upgrade
Il manifest può essere evoluto a:
- Livello OPPONIBILE (IPFS + BTC/ETH)
- Livello FORTE (IPFS + BTC + ETH)
